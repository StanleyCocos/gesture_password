## flutter 手势密码插件

### 简介
  可以定制化的手势密码组件

### 引入
``` dart
dependencies:
  flutter_gesture_pwd: ^1.0.0
```
### 使用
属性 | 说明
------------ | -------------
color | 修改默认圆圈颜色
highlightColor  | 修改选择圆圈颜色
pathColor | 修改连线画笔颜色
frameRadius  | 圆圈中心点半径
pointRadius  | 圆圈半径
pathWidth  | 连线画笔宽度

onFinishGesture  | 结束绘制返回结果(数组下标)
